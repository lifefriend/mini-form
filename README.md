# mini-form
A demo web component that uses Custom Elements v1 and Shadow Dom v1. See a sample online at [https://mini-form-demo.firebaseapp.com/](https://mini-form-demo.firebaseapp.com/).

## installation
```
npm install
bower install
```

## testing
Run web-component-tester
```
> wct
```
